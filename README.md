### Выводим информацию и клиете и сервере Docker
```shell
docker version
```
---

### Выводим информацию о запущенных и остановленных контейнерах
```shell
docker ps -a
```

---
### Выводим список локальных образов
```shell
docker images
```
---
### Создание контейнера
```shell
docker run <container_name>
```
